## Viewing

Viewing the slides locally (for editing, perhaps) requires a HTTP server or developer-mode browser settings. I use

```
$ python2 -m SimpleHTTPServer
```

or

```
$ python3 -m http.server
```

to view them, but do whatever you prefer.
